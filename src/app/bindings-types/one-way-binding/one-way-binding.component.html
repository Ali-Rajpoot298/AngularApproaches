One Way Binding

One-way binding refers to the flow of data from the component (business logic) to the view (template) or from the view to the component, but not both simultaneously.

Inerpolation

Property Binding

Binds a property of a DOM element to a property or expression in the component, allowing you to dynamically set properties. e.g

[src]="imageUrl" src is a DOM element, here we bind src property with imageURl proerpty which is the property define in component

Event Binding

allows you to capture and respond to events raised by the DOM elements in your application. It facilitates communication from the view (template) to the component (business logic).

Events like onclick, onInputchange(), onMouseOver() etc

Used to dynamically set the value of CSS styles based on conditions.
